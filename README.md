# Movies & Games

A modern web platform to discover and access free movies and games in one place. Intuitive interface with attractive design and fluid navigation.

---

## Summary

Movies & Games is a web application that offers a curated collection of free games and movies. With an elegant card-based interface and dark theme, it allows users to explore entertainment content conveniently.

**Main Features:**
- 🎮 Games section with popular titles (Stardew Valley, Minecraft, Hytale)
- 🎬 Catalog of movies and visual content
- 🎨 Responsive design with a modern theme (purple/blue)
- ⚡ Fast and intuitive interface
- 📱 Optimized for different devices

**Technologies Used:**

- HTML5
- CSS3
- JavaScript
- Netlify (hosting)

**How ​​to Use:**
1. Access the platform at [movies-games-free.netlify.app](https://movies-games-free.netlify.app)
2. Browse the Games or Movies section
3. Click on the desired card to access the content
4. Enjoy the free entertainment
